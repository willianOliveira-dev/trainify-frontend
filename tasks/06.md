## Tarefa

Crie um chatbot que chame a rota `/ai/chat` da API (não use Orval).

## Requisitos técnicos

- Use Vercel AI SDK.
  ```tsx
  const { messages, sendMessage, status } = useChat({
    transport: new DefaultChatTransport({
      api: "/ai",
    }),
  });
  ```
- Armazene o estado de abertura do chat na URL e também armazene uma mensagem inicial que será enviada ao chat automaticamente. Use a biblioteca "nuqs" para fazer isso. Exemplo: localhost:3000?chat_open=true&chat_initial_message=message
- Abra o chatbot ao clicar no botão central da barra de navegação ( componente navigation-bar-client.tsx) (com ícone Sparkles).
- Abra o chatbot ao clicar no botão "?" do @exercise-item.tsx e a mensagem inicial (passada via URL) deve ser "Como executar o exercício X corretamente?"
- Use a biblioteca "streamdown" para formatar as mensagens do chatbot.
- Ao abrir o chatbot e não haver nenhuma mensagem, exiba as seguintes mensagens sugeridas: "Monte meu plano de treino".
- Seja 100% FIEL AO FIGMA. Veja https://www.figma.com/design/yQ8SEXsL9ghLG0ewiqQmMa/FIT.AI--Alunos-?node-id=3606-936&m=dev
- Use o componente Drawer do Shadcn para criar o chatbot.
- O drawer deve ter um tamanho de consideravel para a comunicação.
- O drawer deve ter um botão de fechar no canto superior direito.
