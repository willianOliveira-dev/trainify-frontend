## Tarefa

Crie a tela de login e integre ela com a API

## Requisitos Técnicos

- Seja 100% FIEL ao Figma em https://www.figma.com/design/yQ8SEXsL9ghLG0ewiqQmMa/FIT.AI--Alunos-?node-id=3606-1276&m=dev

- Ao clicar em "Fazer login com o Google" chame o src\lib\auth-client.ts para fazer a autenticação.
- Ao acessar a página inicial, caso o usuário não esteja logado, redirecione para a tela de login.
- Ao fazer login, caso o usuário já esteja logado, redirecione o usuário para a página inicial.
